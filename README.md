## ინსტრუქცია 

- არ შექმნათ ბრენჩი ამ რეპოზიტორიაში, დაფორკეთ რეპოზიტორია, თქვენთან შექმენით ახალი ბრენჩი, იქ იმუშავეთ და შემდეგ გამოაგზავნეთ აქ PR.
